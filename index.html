> CHESSTOURNEY - GESTOR DE TORNEOS DE AJEDREZ

Para el proyecto final de mi carrera —la Tecnicatura en Programación de la Universidad Tecnológica Nacional de Córdoba— creé una web app para la gestión de torneos de ajedrez de un modo sencillo y amigable. La aplicación consistía en usuarios de tipo "organizador" y usuarios de tipo "participante". El rol de organizador permitía crear torneos, agregando los detalles necesarios para que los usuarios que quisieran competir fueran capaces de encontrar fácilmente el torneo según las características que prefirieran.


> CARACTERÍSTICAS

  • Buscador de torneos con filtros avanzados
  • Formularios reactivos para la creación y edición de los torneos
  • Visualización interactiva de partidas jugadas
  • Análisis de partidas con Stockfish
  • Dashboard con reportes estadísticos
  • Interfaz adaptativa según el rol de usuario
  • Registro / Inicio de sesión
  • Integración con MercadoPago
  • Interfaz de usuario responsiva

> TECNOLOGÍAS UTILIZADAS

  • SQL Server
  • Angular
  • NET Core
  • Bootstrap
  • Stockfish (Motor de ajedrez)
  • Chessboardjs
  • Chessjs

> GENERADOR DE IDEAS CODING

Este es un generador de ideas de programación basado en la restricción de "tipos" y "temas". El generador simplemente elige aleatoriamente un tipo, que refiere al tipo de proyecto, y un tema, que refiere a una temática general a la que orientar el proyecto. La idea es que se utilice para obtener inspiración para la creación de proyectos, como una forma de brainstorming.


> CARACTERÍSTICAS

  • Gran variedad de temas
  • Gran variedad de tipos
  • Generación instantánea
  • Generar solo tipo o tema
  • Interfaz intuitiva, agradable y responsiva
  • Sección de preguntas frecuentes donde se explica cómo utilizar la aplicación

> TECNOLOGÍAS UTILIZADAS

  • Angular
  • TailwindCSS
  • ElectronJS

> GESTOR DE COMPRA/VENTAS PARA HELADERÍA

Este fue un proyecto para una materia de la universidad, donde trabajamos junto con algunos compañeros creando una aplicación de escritorio. Es un simple gestor de compra de insumos y venta de helados para una heladería ficticia.


> CARACTERÍSTICAS

  • Automatización del cálculo de gastos
  • Automatización de cálculo de ventas
  • Simple generador de factura para la venta
  • Buscador de producto
  • Buscador de Insumo
  • Buscador de clientes regulares con filtro por apellidos
  • Inicio de sesión
  • Interfaz simple y responsiva
  • Base de datos relacional donde se almacenan datos de clientes, insumos para la compra y venta.

> TECNOLOGÍAS UTILIZADAS

  • SQL Server
  • Java
  • Java Swing
  • Librería FlatLaF para Java Swing
